Para comentar se necesitan iniciar con "< !--" y terminar con "-- >"

listas: Existen dos tipos de listas:

Unordered List:

Es una lista donde no importa el orden, como por ejemplo la lista del supermercado

Se crean con "< ul >" y dentro, por cada elemento se debe utilizar "< li >"

Quedando asi:

  • item 1
  • item 2
  • item 3

Ordered List:

Lista dónde si importa el orden de los elementos

Se crean con "< ol >" y dentro, por cada elemento se debe utilizar "< li > "

Quedando asi:

  1. item 1
  2. item 2
  3. item 3
